# 🔍 Plagiarism Hunter Pro

An all-in-one AI content analysis tool that detects plagiarism, identifies AI-generated text, and humanizes content using state-of-the-art NLP models.

## 🚀 Features
- **Plagiarism Detection** — Scans text against multiple sources to detect copied content
- **AI Content Detector** — Identifies whether text was written by AI (GPT, etc.)
- **Text Humanizer** — Uses T5-based paraphrasing to make AI-generated text sound natural
- **Interactive Dashboard** — Clean Streamlit UI with KPI cards for quick insights
- **Cross-platform** — Flutter frontend for mobile/desktop access

## 🛠️ Tech Stack
- **Backend:** Python, Streamlit
- **NLP Model:** T5 (Text-to-Text Transfer Transformer) for humanization
- **Frontend:** Flutter
- **Libraries:** Transformers, PyTorch, Scikit-learn

## ⚙️ How to Run
```bash
git clone <your-repo-link>
cd plagiarism-hunter-pro
pip install -r requirements.txt
streamlit run app.py
```

## 📊 How It Works
1. Upload or paste text
2. System scans for plagiarism and AI-generation markers
3. View results on the KPI dashboard
4. Optionally humanize AI-flagged content
